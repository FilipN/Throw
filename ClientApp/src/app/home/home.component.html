Goal

Our main motive is trying to mix productivity and creativity, giving collegues, fellow students and developers a platform to share and collaborate on python projects. Our idea is to include more and more programming languages and libraries that could be included and imported in user scripts.

Built-in Python compiler

Python is one of the fastest growing programming languages that has applicability in many scientific paths. On Collaborators we are dedicated to make writing python scripts easier. All you have to do is login with your Facebook or Google account and you can write and compile your script on our server.

Interactive editing

If you want to edit your script with some friends or collegues, you can do it here. We can assure you that when you are writting on some line, nobody else can change it. But they can modify any other. When you stop working on it, it is automatically unlocked, and it can be edited by someone else.
